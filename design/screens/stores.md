## 店舗作成（ドロワー）

**Figma**: [店舗作成画面](https://www.figma.com/design/rla6ZV5ACVzUqqWzuaDglx/Auction-App-Prototype?node-id=22351-206955&m=dev)

### レイアウト

- 背景にオーバーレイ（`bg-white opacity-90`）
- 右サイドからスライドインするドロワー形式
- ヘッダー固定（「店舗を作成」タイトル + キャンセル + 作成ボタン）
- フッター固定（キャンセル + 店舗を作成ボタン）
- 中央コンテンツスクロール

### フォームセクション構成

1. **基本情報**
   - 店舗名: テキスト入力（full width）
   - 提供サービス: チェックボックス（買取 / 小売 / 質）

2. **連絡先情報**
   - 住所: テキスト入力（full width）
   - 電話番号: テキスト入力（左列）
   - メールアドレス: テキスト入力（右列）

3. **店舗画像**
   - ドラッグ&ドロップエリア
   - アイコン + 「クリックまたはドラッグして画像をアップロード」
   - キャプション: 「PNG, JPG（最大5MB）」

4. **営業時間設定**
   - 各曜日（月〜日）をリスト表示
   - 曜日ラベル + トグルスイッチ + 開始時間 〜 終了時間
   - トグルOFF時: 「定休日」テキスト表示
   - デフォルト: 月〜金 10:00〜20:00、土 11:00〜19:00、水・日 定休日

5. **所属従業員**
   - テーブル（氏名 / ロール / ステータス）
   - チェックボックスで選択
   - 管理者は `#c9ecdf` 背景バッジ

6. **プライベート設定**
   - ラベル + 説明テキスト + トグルスイッチ（右寄せ）

### 再現チェックリスト

- [ ] 「店舗を作成」ボタンがプライマリボタン（`#095073`）
- [ ] 「キャンセル」ボタンがセカンダリボタン
