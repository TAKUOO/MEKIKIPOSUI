## C8. トースト通知

**Figma参照**: [ストア一覧](https://www.figma.com/design/rla6ZV5ACVzUqqWzuaDglx/Auction-App-Prototype?node-id=22351-206426&m=dev) / [従業員管理](https://www.figma.com/design/rla6ZV5ACVzUqqWzuaDglx/Auction-App-Prototype?node-id=22360-306153&m=dev)

**共通仕様**

| 項目 | 値 |
|---|---|
| 背景色 | `#FFFFFF` |
| border-radius | 12px |
| 幅 | 480px |
| padding | `px-20 py-16` |
| 位置 | 画面右上（コンテンツエリア内、top: 23.74px） |
| gap | 12px |

**成功トースト**

- ボーダー: 1px solid `rgba(77,155,106,0.19)`
- アイコン背景: `#e8f5ed`（40×40px、border-radius: 100px）
- アイコン: 24×24px `check` または `person_check`（白）

**エラートースト**

- ボーダー: 1px solid `rgba(220,71,73,0.25)`
- アイコン背景: `#fdecea`（40×40px、border-radius: 100px）
- アイコン: 24×24px `error`（`#dc4749`）

**テキスト**

- タイトル: 14px SemiBold、`#1a1918`
- 本文: 12px Regular、`#6d6c6a`
- 閉じるボタン: `mdi_close`（18×18px）

**バリエーション（成功）**

| アクション | タイトル | 本文 | アイコン |
|---|---|---|---|
| 店舗作成 | 店舗を作成しました | 新しい店舗が正常に登録されました | `check` |
| 店舗更新 | 店舗情報を更新しました | 店舗情報が正常に保存されました | `check` |
| 従業員追加 | {名前}さんを追加しました | 新しい従業員が正常に登録されました | `person_check` |
| 従業員更新 | {名前}さんの情報を更新しました | 従業員情報が正常に保存されました | `person_check` |

**バリエーション（エラー）**

| アクション | タイトル | 本文 | アイコン |
|---|---|---|---|
| 店舗作成失敗 | 店舗の作成に失敗しました | 入力内容を確認して、もう一度お試しください | `error` |
| 店舗更新失敗 | 店舗情報の更新に失敗しました | 入力内容を確認して、もう一度お試しください | `error` |
| 従業員追加失敗 | 従業員の追加に失敗しました | 入力内容を確認して、もう一度お試しください | `error` |
| 従業員更新失敗 | 従業員情報の更新に失敗しました | 入力内容を確認して、もう一度お試しください | `error` |
